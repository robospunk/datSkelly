Font size

If you'd like to change the default font size, you can add these classes:

Add the class "tsm" for small text.

Add the class "treg" for normal 1rem text.

Add the class "tmd" for medium text.

Add the class "tlg" for large text.

Add the class "txl" for extra large text.

Font weight and decoration

If you'd like to change the weight or decoration, use these clases:

Add the class "wlight" for light weight text.

Add the class "wreg" for normal weight text.

Add the class "semi" for semi-bold text.

Add the class "bold" for bold text.

Add the class "italic" for italicized text.

Add the class "underline" for underlined text.

Buttons

Buttons and anchor tags are styled to look the same.

By default, they do not have a background color or outline.

<a> <a class="btn">

You can style buttons individually with the class "btn"

Or in groups "btns"

<div class="btns"> <a>button 1</a> <a>button 2</a> <a>button 3</a> </div>

The code above would output this:

button 1 button 2 button 3

By default, the class "btn" makes the button have a primary background color.
Here are a few other options:

class of "btn" class of "btn success" class of "btn danger"

navigation

Standard Nav

The standard nav without any classes is set to display: flex
It will disperse it's first children evenly using justify-content: space-between

<nav class="primary"> <a>link1</a> <a>link2</a> <a>link3</a> </nav>

Outputs

link1 link2 link3

If you want to cluster buttons, you can wrap button groups in divs with a class of "flex"

<nav class="primary"> <a>Home</a> <div class="flex"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </nav>

Outputs

Home
link1 link2 link3

Collapsable Nav

To have a menu that collapses at small screen widths, wrap your links in a div with the class of "collapse" and make sure the nav contains a button with the class of "collapseToggle"

<nav class="primary"> <a>Home</a> <button class="collapseToggle">Menu</button> <div class="collapse"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </nav> <!-- OR --> <div class="nav primary btns"> <a>Home</a> <button class="collapseToggle">Menu</button> <div class="collapse"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </div>

Grid

The grid is flex based and uses 10 columns. Screens with a width 580px or smaller will ignore grid rules.

To start a grid, make a div with a class of "row"

The direct children of this div will be affected by grid rules. By default, children of "row" will distribute space horizontally and evenly if space allows for it. If there is not enough space, chidlren will wrap to the next line.

<div class="row"> <div class="border">1</div> <div class="border">2</div> <div class="border">3</div> <div class="border">4</div> <div class="border">5</div> <div class="border">6</div> <div class="border">7</div> <div class="border">8</div> <div class="border">9</div> <div class="border">10</div> </div>

To change the column width, add one of the following classes to the direct child of "row":

"col10" "col20" "col30" "col40" "col50" "col60" "col70" "col80" "col90" "col100"

The number of each class is associated with the percentage of space it will take up, if there is enough space.

Checkbox and Radio

If you want your checkbox or radio inputs to display next to its label, you must wrap it in a div with the class of "check" or "radio".

The classes check and radio technically do the same thing, therfore they are interchangeable

<div class="check"> <input type="checkbox" value="check1" name="check"> <label>Checkbox 1</label> </div> <div class="check"> <input type="checkbox" value="check2" name="check"> <label>Checkbox 2</label> </div> <div class="check"> <input type="checkbox" value="check3" name="check"> <label>Checkbox 3</label> </div>

Outputs

<div class="check"> <input type="radio" value="radio1" name="radio"> <label>Radio 1</label> </div> <div class="check"> <input type="radio" value="radio2" name="radio"> <label>Radio 2</label> </div> <div class="check"> <input type="radio" value="radio3" name="radio"> <label>Radio 3</label> </div>

Advanced Field Layout

If you'd like to organize fields further, you can place them in rows or in divs with the class of "flex".

<div class="flex"> <div class="check"> <input type="checkbox" value="check1" name="check"> <label>Check 1</label> </div> <div class="check"> <input type="checkbox" value="check2" name="check"> <label>Check 2</label> </div> </div>
